##사용 전 주의사항##

1. 작업관리자 권한으로 실행하셔야합니다.
2. 모든 이미지파일 확장자는  " 참가자 XXXX.jpg "  로 통일하셔야합니다 (png,jpeg 등 형식은 인식안됨, 이름 띄우고 전화뒷자리.jpg 형식입니다. )
3. 작업 폴더 안에는 다른 중요문서를 같이 두시지 않는걸 권장합니다.


##사용방법

1.엑셀에서 회원명과 전화번호를 연속된 2열로 복사해서 xxlname.txt에 붙여넣어주세요. 

ex) A열=이름 , B열 = 전화번호      시작지점부터 바꾸고자 하는 명단까지 복사

	> 이미 변경완료된 회원분은 xxlname.txt 파일에서 이름을 빼주셔야합니다.

2.py01po.exe파일을 관리자 권한으로 실행합니다.

ex)py01po.exe파일을 우클릭해서 관리자권한으로 실행하기

	> 만약, 일반 더블클릭으로 실행한다면 작동하지 않습니다.

2.xxlnmae.txt의 첫번째 오는분에게 부여하고자하는 넘버를 알파벳과 숫자를 구분해서 프로그램에 작성해주세요.

ex)알파벳 : K  ,  숫자 : 3000

	> 오름차순으로 숫자가 올라가면서 값이 부여됩니다. K3000 ~

3.실행 버튼을 눌러주시면 작업이 시작됩니다.

	> 완료가 되었다면 마지막자리를 확인해보세요.


## : 오류가 떴다면 회원넘버와 성함을 xxlname.txt파일과 폴더의 바뀌지않은 사진이름을 확인해보세요.

## : 작업 중단 후 다시 작업하실때는 xxlname.txt파일도 다시 작성해주세요.
